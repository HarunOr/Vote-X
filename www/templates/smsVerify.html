Verifizierung
Geben Sie ihre Handynummer ein
Sie erhalten einen Code per SMS, mit dem Sie sich verifizieren können
Ihre Nummer muss mit der deutschen Ländervorwahl beginnen (49).
Ihre Nummer muss eine gültige Länge haben.
Ihre Nummer darf nicht mit einer 0 beginnen.
Beispiel: 4915212345678
Geben Sie jetzt den Code ein, der per SMS an ihre Nummer versendet wurde
